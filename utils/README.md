# Utils

Shared utility modules providing foundational functionality for the BRI database system.

## Overview

This directory contains standalone utility modules that are used across the storage, engine, and client layers. Each module is self-contained with no dependencies on other BRI modules.

## Modules

- **jss/** - JSON SuperSet serialization with extended type support
- **diff/** - Object change tracking and path operations
- **schema/** - Schema validation with type checking
